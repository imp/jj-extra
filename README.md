# jj-extra - Jujutsu CLI extra aliases

This plugin provides additinal aliases for [jj](https://jj-vcs.github.io/jj).

To use it, add `jj-extra` to the plugins array of your zshrc file:

```zsh
plugins=(... jj-extra)
```

## Aliases

| Alias  | Command                       |
| ------ | ----------------------------- |
| jjgpN  | `jj git push --allow-new`     |
| jjsh   | `jj show`                     |
